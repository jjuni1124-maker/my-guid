# 필수 준비물

학교 태블릿

웹 브라우저(Chrome 등)만 있으면 지금 바로 GitHub 저장소를 만들고 관리할 수 있습니다.

GitHub 계정

나의 코드를 백업하고 전 세계 개발자들과 공유하기 위해 반드시 계정이 필요합니다.

GitHub 회원가입 따라하기

처음 보는 영어 화면도 이 순서대로만 따라오세요!

01

github.com 접속 후 오른쪽 위 Sign up 버튼을 누르세요.

02

Email 입력: 학교나 개인 이메일을 입력하고 "Continue"를 누릅니다.

03

Password 설정: 8자 이상의 강력한 비밀번호를 만드세요.

04

Username 결정: 나만의 닉네임을 만드세요. (중복 불가)

05

인증 확인: 퍼즐을 풀고 이메일로 온 8자리 숫자를 입력하면 가입 완료!

Git 이란?

내 컴퓨터 안의 '타임머신'입니다. 코드를 작성하다가 실수했을 때, 혹은 예전 상태로 되돌리고 싶을 때를 대비해 모든 변화 과정을 기록하는 도구입니다.

GitHub 란?

Git으로 기록한 코드들을 모아두는 '온라인 금고'이자 개발자들의 'SNS'입니다. 내 작업물을 백업하고 다른 사람과 함께 협업할 수 있는 웹 서비스입니다.
